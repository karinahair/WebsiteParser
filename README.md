# Парсер новостей с сайта Известия

## О проекте
Этот скрипт парсит новости с архивной версии сайта "Известия". Он собирает заголовки новостей и ссылки на них, затем группирует новости по разделам.

## Что делает программа
- Заходит на сайт с новостями
- Находит все новостные блоки
- Извлекает названия разделов, заголовки новостей и ссылки
- Группирует новости по разделам
- Выводит результат в консоль

## Установка зависимостей
pip install -r requirements.txt
